# Contributing to SMTP Router

Thank you for your interest in contributing! We welcome bug reports, feature requests, and pull requests.

## Getting Started

1.  **Fork** the repository.
2.  **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/smtp-router.git`
3.  **Install dependencies**: `go mod download`

## Development

- Run the server: `go run .`
- Run tests: `go test ./...`

## Pull Requests

1.  Create a new branch: `git checkout -b feature/my-feature`
2.  Make your changes.
3.  Add tests if applicable.
4.  Commit your changes: `git commit -m "Add my feature"`
5.  Push to your fork: `git push origin feature/my-feature`
6.  Open a Pull Request.

## Code Style

- Follow standard Go conventions (run `go fmt ./...`).
- Keep Starlark scripts simple and readable.
